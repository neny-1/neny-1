## 🚀 What I Build

### 🔭 AiTech Safety Monitor — Live Production System
> Computer vision platform for real-time worksite safety monitoring

- 7-plugin event-driven architecture processing **6+ simultaneous camera streams at 15 FPS**
- Covers PPE detection, fire & smoke, occupancy zones, dangerous zone access, and activity tracking
- Real-time **WebRTC streaming** infrastructure with Redis message brokering
- Multi-object tracking with **DeepSORT / ByteTrack**
- Automated alerts via Telegram, SMS, and email with snapshot capture
- Containerized with **Docker**, integrated into Laravel backend via WebSockets

🔗 [Live Platform](https://aitech.net.au/solutions/aitech-safety)

---

### 🧠 Multimodal OCR & VLM Document Intelligence Tool
> AI agent node for structured document extraction

- Hybrid pipeline: **PaddleOCR** + **Qwen-VL** (Hugging Face) as VLM fallback
- Handles images, scanned PDFs, and Word documents with embedded visuals
- Designed as a node within a larger multi-agent pipeline

---

### 💬 Medical Chatbot using RAG
> Production-ready retrieval-augmented generation app

- Ingests documents → vector knowledge base → context-aware API responses
- Stack: Python, FastAPI, Docker, OpenAI, Cohere, MongoDB, vector databases
